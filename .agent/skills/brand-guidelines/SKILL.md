# Flexbike Brand Styling

## Overview

To access Flexbike's official brand identity and style resources, use this skill. Flexbike is a motorcycle rental marketplace connecting riders with verified rental companies.

**Keywords**: branding, corporate identity, visual identity, post-processing, styling, brand colors, typography, Flexbike brand, motorcycle rental, visual formatting, visual design

## Brand Guidelines

### Colors

**Main Colors:**

- Primary: `#08b7b7` - Teal accent and branding
- Dark: `#242420` - Primary text and dark backgrounds
- Eggshell: `#F6F5F1` - Light backgrounds and text on dark
- Dark Eggshell: `#EBE8E1` - Secondary elements and subtle backgrounds

**Accent Colors:**

- Yellow: `#FFE450` - Secondary accent from logo variations
- Blue: `#0066CC` - Tertiary accent for highlights
- Charcoal: `#2D2D2A` - Alternative dark for contrast

### Typography

- **Headings**: Nunito (with Arial fallback)
- **Body Text**: Nunito (with sans-serif fallback)
- **Note**: Nunito fonts are available in the project fonts directory

## Features

### Smart Font Application

- Applies Nunito font to all text elements
- Uses appropriate font weights (Bold, SemiBold, ExtraBold, Black)
- Automatically falls back to Arial/sans-serif if custom fonts unavailable
- Preserves readability across all systems

### Text Styling

- All text: Nunito font family
- Smart color selection based on background
- Preserves text hierarchy and formatting
- Optimized for motorcycle rental marketplace branding

### Shape and Accent Colors

- Non-text shapes use Flexbike accent colors
- Cycles through teal, yellow, and blue accents
- Maintains visual interest while staying on-brand
- Reflects motorcycle and adventure theme

## Technical Details

- Uses system-installed Nunito fonts when available
